# Whale Shark Project

The Whale Shark Project is a sustainable financial model for nano-business. This README file will act as an outline for development.

Mission Statement:

This project aims to bridge the gap between living paycheck to paycheck and building wealth using a sustainable passive income.

Using a solid foundation in traditional finances this project offers a money management approach that does not pull funds from a main income that supports necessary expenses. 

With interest rates bottoming out it is more important now then ever to create business on a thin margin. This project will also offer guidance to maximize interest rates using a mixed portfolio of traditional investments and decentralized finance.

The principal of the philosophy is that every dollar counts. This project aims to change the perspective of high risk entrepreneurship to that of low risk wealth building. 

A nano-ecenomic version of Warren Buffets Permenant Capital Vehicle (PCV) accessible to the lay investor.
